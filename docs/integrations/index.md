# Intégrations

Cette section liste les intégrations Home Assistant que j’utilise (ou que je
prévois d’utiliser) dans SmartHub, avec un lien vers leur documentation.

## Intégrations utilisées

- **Météo-France**
  - Type : natif
  - Source : https://www.home-assistant.io/integrations/meteo_france/
  - Rôle : apporter la météo et les prévisions (avec un focus France) dans Home
    Assistant.
- **Météo-France Cards**
  - Type : HACS
  - Source : https://github.com/hacf-fr/lovelace-meteofrance-weather-card
  - Rôle : ajouter des cartes de tableau de bord pour afficher la météo.
- **Spook**
  - Type : HACS
  - Source : https://spook.boo
  - Rôle : ajouter une boîte à outils et des aides au diagnostic pour Home
    Assistant.

::: info
Je garde cette section volontairement courte. Si une intégration mérite une
explication plus détaillée dans le contexte de mon SmartHub, j’ajouterai ensuite une
page dédiée.
:::
